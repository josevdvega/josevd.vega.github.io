Utilización de React

¿Cómo empezar a crear una aplicación base?

Ahora que tienes React instalado y listo para usar, ¡vamos a crear tu primera aplicación!

Paso 1: Asegúrate de estar en la carpeta del proyecto

Abre la terminal y navega hasta la carpeta de tu proyecto si no estás ahí. Por ejemplo, si tu proyecto se llama mi-primera-app, usa el siguiente comando:

cd mi-primera-app

Paso 2: Edita el archivo App.js

El archivo App.js es como el "corazón" de tu aplicación, ya que es el punto donde se construye la interfaz principal. Para editar este archivo:

  • Abre la carpeta src dentro de tu proyecto.
  • Busca el archivo llamado App.js.
  • Ábrelo en el editor de texto de tu preferencia (por ejemplo, Visual Studio Code).

Paso 3: Escribe tu primer código de React en App.js

Dentro de App.js, verás algunas líneas de código que generan la interfaz básica. Puedes modificar el contenido de la función App para mostrar un mensaje personalizado:

function App() {
    return (
        

            
¡Hola, Mundo en React!

            
Esta es mi primera aplicación de React.

        

    );
}

export default App;

Este código cambiará la página inicial para mostrar el mensaje ¡Hola, Mundo en React! y un párrafo.

¿Cómo visualizar los resultados de la ejecución?

Para ver tu aplicación en el navegador, asegúrate de estar en la carpeta de tu proyecto en la terminal y escribe el siguiente comando:

npm start

Este comando abrirá automáticamente una ventana en tu navegador y mostrará tu aplicación. Cualquier cambio que hagas en App.js se reflejará en el navegador una vez que guardes el archivo.

Consejo:

Si no se actualiza automáticamente, puedes actualizar la página en tu navegador para ver los cambios. También puedes usar la combinación de teclas Ctrl + R (Windows) o Cmd + R (Mac).
