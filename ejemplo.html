Ejemplo Avanzado: Reloj en Tiempo Real

En este ejemplo, vamos a crear un reloj que muestra la hora actual y se actualiza automáticamente cada segundo. Esto nos ayudará a entender cómo funcionan los componentes en React y cómo podemos hacer que se actualicen en tiempo real.

Paso 1: Crear el Componente Reloj

Primero, crearemos un nuevo componente llamado Reloj.js. Este componente será responsable de mostrar la hora actual.

import React, { useState, useEffect } from 'react';

function Reloj() {
    const [hora, setHora] = useState(new Date().toLocaleTimeString());

    useEffect(() => {
        const intervalo = setInterval(() => {
            setHora(new Date().toLocaleTimeString());
        }, 1000);

        // Limpieza del intervalo cuando el componente se desmonta
        return () => clearInterval(intervalo);
    }, []);

    return (
        

            
Hora Actual: {hora}

        

    );
}

export default Reloj;

¿Qué hace este código?

  • Importamos useState y useEffect de React para manejar el estado y los efectos secundarios.
  • Inicializamos el estado hora con la hora actual.
  • Usamos setInterval dentro de useEffect para actualizar la hora cada segundo.
  • Finalmente, mostramos la hora en el componente.

Una vez que hayas hecho esto, guarda todos los archivos y ejecuta tu aplicación con npm start. Deberías ver el reloj mostrando la hora actual, actualizándose cada segundo.

Consejos Finales

Intenta experimentar con el componente. Por ejemplo, puedes agregar más estilos, cambiar el formato de la hora, o incluso agregar la fecha. Esto te ayudará a comprender mejor cómo funciona React y cómo puedes manipular el estado en tus aplicaciones.
