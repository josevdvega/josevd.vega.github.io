Primeros Pasos con React

Tu primer componente: ¡Hola Mundo!

En React, todo comienza con los componentes. Los componentes son bloques de construcción que crean la interfaz de usuario. En este primer ejemplo, crearemos un simple componente que muestra un mensaje: "¡Hola, Mundo!"

Paso 1: Encuentra y abre el archivo App.js

Dentro de la carpeta de tu proyecto, verás varias subcarpetas y archivos. Busca la carpeta llamada src y dentro de ella encontrarás el archivo App.js.

Para abrir el archivo, puedes usar cualquier editor de texto, como Visual Studio Code, Atom o incluso el Bloc de Notas. Si estás usando Visual Studio Code, haz clic derecho en App.js y selecciona "Abrir con Visual Studio Code".

Paso 2: Crea un componente en React

Con el archivo App.js abierto, verás algo de código que puedes reemplazar por el siguiente:


// App.js

import React from 'react';

function App() {
    return (
        <div>
            <h1>¡Hola, Mundo!</h1>
        </div>
    );
}

export default App;

Paso 3: Ejecuta el proyecto

En la terminal, ejecuta el siguiente comando para iniciar tu proyecto:

npm start

Esto abrirá tu proyecto en el navegador y verás el mensaje "¡Hola, Mundo!" en la pantalla. ¡Felicidades! Has creado tu primer componente en React.

¿Qué aprendimos?

Este simple ejemplo muestra cómo React toma componentes y los convierte en elementos visuales en la página. Puedes pensar en un componente como una "pieza" del sitio web que puedes reutilizar en diferentes partes.
