Explicación del Funcionamiento del Framework

¿Sobre qué actúa React?

React actúa principalmente sobre la interfaz de usuario (UI) de una aplicación web. Esto incluye elementos como botones, textos, imágenes y cualquier otra parte que veas en la pantalla. React permite crear componentes, que son bloques de construcción que puedes reutilizar en diferentes partes de tu aplicación.

¿Cómo se editan y modifican los archivos que actúan en el framework?

Para editar los archivos de tu aplicación React, simplemente necesitas abrir el archivo correspondiente en tu editor de texto. Por ejemplo, para modificar lo que se muestra en la pantalla, puedes editar App.js como mencionamos anteriormente. Aquí hay algunos pasos básicos para editar archivos:

  • Abre tu editor de texto (como Visual Studio Code).
  • Navega hasta la carpeta src de tu proyecto.
  • Haz doble clic en el archivo que deseas editar, por ejemplo, App.js.
  • Realiza los cambios que desees y guarda el archivo.

¿Cómo se referencian los archivos creados desde otras aplicaciones?

Cuando creas una nueva aplicación en React, cada componente que creas se guarda en su propio archivo. Por ejemplo, si creas un nuevo componente llamado Saludo.js, necesitarás importar este componente en App.js para que pueda ser utilizado. Aquí te mostramos cómo hacerlo:

import Saludo from './Saludo'; // Esto importa el componente Saludo

Después de importarlo, puedes utilizar el componente dentro de la función App como si fuera una etiqueta HTML:

function App() {
    return (
        

             {/* Aquí estamos usando el componente Saludo */}
        

    );
}

Consejo:

Es importante asegurarse de que el nombre del archivo y el nombre del componente coincidan. Por ejemplo, si tu archivo se llama Saludo.js, el componente dentro de ese archivo debe llamarse Saludo.
