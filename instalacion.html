Instalación de React

¿Se necesita instalación?

Sí, React necesita instalarse antes de poder utilizarlo. Para ello, vamos a seguir unos pasos detallados que asegurarán que tu equipo esté preparado para crear proyectos en React.

Paso 1: Instalar Node.js

React se basa en Node.js y su administrador de paquetes, npm. Para instalar Node.js, visita la página oficial: https://nodejs.org/ y selecciona la última versión recomendada. Sigue las instrucciones de instalación del instalador para tu sistema operativo.

Paso 2: Crear una carpeta de proyectos

Para mantener tus proyectos organizados, crea una carpeta en tu equipo donde guardarás cada uno de los proyectos que realices en React. Puedes crear esta carpeta en cualquier ubicación; te sugerimos llamarla ReactProjects en un lugar fácil de recordar, como en el escritorio o en la carpeta Documentos.

Paso 3: Crear un nuevo proyecto de React

Abre la terminal (o símbolo del sistema) y navega hasta la carpeta ReactProjects. Para hacer esto, usa el siguiente comando en la terminal:

cd [ruta_a_tu_carpeta_ReactProjects]

Una vez que estés dentro de la carpeta, escribe el siguiente comando para crear un nuevo proyecto:

npx create-react-app nombre-del-proyecto

Reemplaza nombre-del-proyecto con el nombre que quieras darle a tu proyecto. Este comando creará una nueva carpeta con el nombre de tu proyecto dentro de ReactProjects. Todos los archivos y configuraciones de React se guardarán en esta carpeta específica.

Paso 4: Entender la estructura del proyecto

Después de la instalación, dentro de la carpeta del proyecto encontrarás varias subcarpetas y archivos:

  • public: Contiene archivos públicos que el navegador puede ver directamente.
  • src: Aquí se encuentran los archivos principales de tu aplicación. Es donde está App.js, el archivo que usaremos como base.
  • node_modules: Contiene las dependencias (bibliotecas) que React necesita para funcionar. No es necesario modificar esta carpeta.

Paso 5: Configurar variables de entorno

Para la mayoría de los proyectos de React, no necesitarás configurar variables de entorno al inicio. Sin embargo, a medida que avances, es posible que desees almacenar información privada, como claves de API, en un archivo especial llamado .env en la raíz del proyecto.

Paso 6: Iniciar el proyecto

Para asegurarte de que todo está funcionando, navega hasta la carpeta de tu proyecto (si no estás allí ya) con el comando:

cd nombre-del-proyecto

Luego, inicia el proyecto con el siguiente comando:

npm start

Este comando abrirá automáticamente tu aplicación en el navegador y mostrará una página de inicio de React, lo cual indica que la instalación fue exitosa.
